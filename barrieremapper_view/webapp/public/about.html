Über BarriereMapper

Der BarriereMapper ermöglicht es, Barrieren auf einer Karte zu markieren. Die erfassten Barrieren können als Karte oder Liste dargestellt werden. Anhand von Beschreibungstexten zu den Barrieren kann ein verbessertes Verständnis der dokumentierten Barrieren entwickelt werden, um darauf geeignete Strategien für die Zugänglichkeit im Raum Heidelberg aufzubauen. Die hier vorliegende prototypische Webanwendung ist im Rahmen einer Masterarbeit im hochschulübergreifenden Studiengang Digitale Methodik in den Geistes- und Kulturwissen entstanden. Weitere Informationenen zu den Inhalten dieser Masterarbeit finden Sie via GitLab Pages.

Anleitung

  1. Erster Schritt: Barriere erfassen
    Um eine Barriere zu erfassen, wählen Sie im Menü den Reiter Barriere aus. Anschließend werden Sie über eine Meldung gebeten, Ihren Standort zur Markierung der Barriere freizugeben oder eine manuelle Markierung per Klick zu setzen. Die erfasste Barriere können Sie dann benennen, beschreiben und optional Ihren Usernamen angeben. Ist der Eintrag Ihrer Barriere erfolgt, erhalten Sie eine Rückmeldung.
  2. Zweiter Schritt: Kartenansicht nutzen
    Wollen Sie eine Karte aller erfassten Barrieren angezeigt bekommen, wählen Sie im Menü den Reiter Kartenansicht aus. Dort wird Ihnen eine Karte der Stadt Heidelberg mit allen erfassten Barrieren angezeigt.
  3. Dritter Schritt: Listenansicht nutzen
    Wollen Sie eine Liste aller erfassten Barrieren angezeigt bekommen, wählen Sie im Menü den Reiter Listenansicht aus. Dort wird Ihnen eine Liste mit allen erfassten Barrieren angezeigt. Diese Liste können Sie sortieren, filtern oder beispielsweise nach Ihrem Username durchsuchen.
